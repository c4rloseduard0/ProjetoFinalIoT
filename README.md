## Objetivo
Este projeto visa automatizar a alimentação de animais domésticos usando recursos online e bluetooth

## Material Utilizado

* 1 Servo Motor 9g
* 1 sensor ultrasônico HC-SR04
* 1 Sensor de temperatura LM35
* 2 Sensor de luminosidade LDR
* 1 ESP32
* 2 Resistores 10k Ω
* 1 Protoboard
* Alguns jumpers
